# 🎓 Smart Attendance Management System

A full-stack web application built using Flask and MongoDB to automate classroom attendance management with secure role-based authentication.

---

## 🚀 Features

### 🔐 Authentication
- User Registration (Student / Faculty / Admin)
- Secure Login with JWT
- Password hashing using bcrypt
- Role-based access control

### 👨‍🏫 Faculty
- Create new classes
- Generate unique Class IDs

### 👨‍🎓 Student
- View available classes
- Mark attendance digitally
- Prevent duplicate attendance
- View attendance status

### 👨‍💼 Admin
- View total users
- View total classes
- View total attendance records

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Flask (Python) |
| Database | MongoDB |
| Authentication | Flask-JWT-Extended |
| Frontend | HTML + Bootstrap |
| Security | bcrypt |
| Version Control | Git & GitHub |
